# HTML & CSS Cheat Sheet

This is my **first project** created during the **Codecademy Front-End Developer** course.

A beautiful, interactive, and practical web development cheat sheet to help reinforce and quickly reference core HTML and CSS concepts.

!([HTML & CSS Cheat Sheet Preview](https://github.com/reinevannel/web-cheatsheet/commit/2166d8ee3fc2dfca58be1a493b48c88cead447a5))  

## ✨ Features

- **Clean & Modern Design** with light/dark mode
- **Live Search** – find tags, properties, or concepts instantly
- **Interactive Tabs** (HTML / CSS)
- **Copy to Clipboard** buttons on every code example
- **Print-friendly** version
- **Smooth animations** + particle background
- **Fully Responsive**

## 🛠 Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Modern techniques)
- Vanilla JavaScript
- Google Fonts (Inter, Play, JetBrains Mono)

## 📚 What I Learned

- Structuring large HTML documents
- Advanced CSS (variables, modern layout, responsive design, print styles)
- DOM manipulation and event handling
- Building interactive UI components
- Writing clean, maintainable JavaScript
- Accessibility basics (ARIA roles, keyboard navigation)
